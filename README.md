# odin-recipes

The aim here is to put the HTML knowledge gained to practice building a basic recipe site. The core of it will be the main index page which will carry links to some recipes.

The projects, which consisted of creating three recipes, was undertaken
successfully. Using knowledge learned upto now, including creating boilerplates, git commiting messages, inserting absolute and relative links, as well as images, using paragraphs, headings, character sets, attributes, entities, and git pushing work, I have created recipes for Jamaican Escovitch
Fish, Ndole, and Australian Lamingtons.

The site was created as follows:
*the main HTML file, index.html, in the root directory, odin-recipes;
*the three recipe files, jamaican-escovitch fish.html, ndole.html, and 
australian-lamingtons.html, in a subdirectory, recipes.
*the images of the recipes in their respective subdirectories named after
them in another subdirectory named images.
